* The User:
Uses the app and interact with it to perform a certain actions.

* The Backend:
Receives requests to perform a business logic which return data or update states.

* The Database:
Write a new data to the database and read from it.

* Third-Party auth services:
Allow the user to authenticate using a third-party services (Google & Apple).
