# Functional Requirements
* The app should allow users to create a new account using their email or a third-party service (e.g., Google or Apple).
* The app should allow users to log in to their existing accounts.
* The app must require users to verify their email upon first registration.
* The app should allow users to update their account information (e.g., password and name).
* The app should allow users to create new bucket lists.
* The app should allow users to modify or delete existing bucket lists.
* The app should allow users to add new items to existing bucket lists.
* The app should allow users to modify or delete items within any existing bucket list.
* The app should provide two tabs for each bucket list: one for current items and one for completed items.
* The app should set the default state of any newly added item to "Current."
* The app should allow users to mark items as "Done."
* The app should allow users to move items from the "Done" state back to the "Current" state.
* The app should allow users to add description per item.
* The app should allow users to mark the priority of each item (from 1 to 10).
* The app should allow users to reorder the items within the any existing bucket list.
* The app should allow users to create shared bucket lists that other users can access and modify.
* The app must be usable offline or online.
* The app must allow users to delete their accounts.
